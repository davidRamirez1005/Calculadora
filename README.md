# Calculadora

Calculadora con solo php

funciona presionando los botones de los numeros y de los signos, el boton con "C" es para borrar todo el contenido en el input y limpiar la session, el boton de "=" es para hacer el calculo y mostrar el resusltado en el input, el boton "‚Üê" es para retroceder o borrar numero por numero.

Asegurese de presionar los botones, no funciona si escribe la operacion con el teclado o la escribe en el input.
